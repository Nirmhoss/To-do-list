Brief Description

This bachelor's thesis is focused on developing a RESTful API for task management (To-Do List) with JWT (JSON Web Token) authentication. The developed API allows users to register, log in, and manage their tasks through secure endpoints. The work is based on modern web development technologies and demonstrates a complete cycle of API creation.

Keywords:

  • REST API
  • JWT Authentication
  • Python
  • Flask
  • SQLite
  • SQL Alchemy

Research Objective

The objective of this bachelor's thesis is to develop a reliable, scalable, and secure API for task management that demonstrates modern approaches to web server development and user authentication.

Main Tasks:

  1. Research and compare modern approaches to RESTful API development
  2. Implement a user registration and authentication system using JWT tokens
  3. Develop CRUD operations for managing user tasks
  4. Implement data validation and error handling at all API levels
  5. Create API documentation and deploy it for demonstration

Research Methodology

During the development of this bachelor's thesis, several methodological approaches were applied:

  • Analysis of existing solutions in the field of task management API development
  • Designing system architecture following REST principles
  • Development based on the Flask microframework using modern patterns
  • Testing the developed API using automated tests and Postman
  • Evaluation of the performance and security of the implemented solution
